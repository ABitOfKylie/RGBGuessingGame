What Is RGB and How To Play

RGB (red, green, and blue) refers to a system for representing the colors used on a computer display. Red, green, and blue can be combined in various proportions to obtain any color in the visible spectrum. Each parameter (red, green, and blue) defines the intensity of the color and can be an integer between 0 and 255. Click image below for more detailed information.


How to Play Each time you choose new colors, a new RGB number will appear in the header. Click on the color square that you think is the correct answer.
Hint: RGB ( 255, 0, 0) would be a bright red color.
